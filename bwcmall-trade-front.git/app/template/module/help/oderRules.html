订单状态
订单状态
订单状态分为五种：“待付款”、“待发货”、“待收货”、“已完成”、“已取消”。各自的含义如下： 待付款

订单已提交成功，等待客户付款。


待发货

订单正在处理中，等待发货。


待收货

货物已经安排发出，等待客户确认收货。


已完成

订单已圆满完成。


已取消

订单已取消，交易已关闭。
