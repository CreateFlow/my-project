取消订单

如果您需要取消订单，在我们还没有确认发货前，您可以登录账号，在订单中心手动取消订单；
当我们还未发货时，并且订单状态为：“待发货”时，您可以联系我们取消订单；
当订单状态变为：“待收货”时则不能再取消订单了，如您需要则在收货后发起退货。
