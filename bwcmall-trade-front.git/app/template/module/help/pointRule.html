仓豆使用说明
定义

1.公司：指上海佰万仓网络科技有限公司。

2.商品：指公司通过网站、微信、APP、宣传手册等方式对外出售和提供的商品。

3.用户：指通过网站、微信、APP、宣传手册等方式了解了公司商品信息，并希望购买或已经购买公司商品的用户。

4.仓豆：指用户在购买公司的商品并支付货款后，根据公司公布的每件货品的仓豆比率所获得的可以在以后的购买中冲抵货款的仓豆；以及公司进行特别活动时授予用户的促销仓豆或奖励仓豆（以下统称"特别仓豆"）。

5.仓豆商品：购买此商品可产生相对应的仓豆。

仓豆点数的确定

1.公司根据商品售价和相关费用为每件商品确定一个仓豆点数。同一商品的仓豆点数得以因市场因素的变化而随时变动。未标明仓豆点数或仓豆点数为 0 的商品为不可获得仓豆的商品。

仓豆点数的计算

1.公司授予仓豆以用户的付款行为为前提。用户可通过公司网站查询订单信息，订单支付完成后，按照用户所购商品的仓豆点数授予相应仓豆，并记录在用户信息中。

2.用户可以通过公司网站(www.bwcmall.com)或客户服务热线(400-021-9898)对自己的仓豆情况进行查询。用户多次购买商品所获得的仓豆可以累计计算。

仓豆有效期

1.用户截止当年6月30日前购买商品所获得仓豆的有效期为当年度的12月31日，下半年7月1日后购买商品所获得仓豆的有效期为次年的6月30日。（注：如用户2016年1月1日获得的仓豆，有效期截止2016年12月31日；如在2016年7月31日获得的仓豆，有效期截止2017年6月30日）。

仓豆的使用

1.用户获得的累计仓豆可以在日后再次购买公司商品时冲抵货款。当次交易所获得的仓豆不能当次使用。每1点仓豆的支付力相当于人民币1元。

2.仓豆可以单独用以支付货款，也可以和公司允许的其他结算方式（如网上付款等）一并使用。

3.用户在一次购买一件商品时方可使用仓豆。

4.公司在接到用户开票通知后，开具发票的金额为用户实际支付的金额

退、换货情况下的仓豆累计问题

用户在对公司出售或提供的商品质量存在异议，要求公司进行退、换货时，其仓豆将根据不同情况作出响应处理。由于订单支付时可能采取现金+仓豆的支付方式，当用户发起退换货时，可能会出现如下两种情况：

1.当退货商品金额小于等于原订单实际支付金额时，公司将根据退换货商品收回原先授予的仓豆。当用户仓豆池数量不足以支付收回仓豆时，将按比列折算现金，从退款中扣取。

2.当退货商品金额大于原订单实际支付金额时，公司将只退还用户实际支付的金额，用户支出和被授予的仓豆将根据退货商品进行相应换算，仓豆池不足部分将相应扣取退款。

3.因商品质量或其他原因而引起的同型号同规格商品的调换不影响仓豆的授予和使用。

仓豆的依据和仓豆的异议

1.公司以用户订单作为每个用户仓豆累计和使用的依据。

2.若用户认为公司对其仓豆累计和使用情况的记载存在错误，可以通过公司网站或客户服务热线向公司提出异议。

3.公司在收到用户关于仓豆异议后，应当立即对该用户的仓豆累计和使用情况予以核实，并在 3 个工作日内作出处理决定并告知用户。

其他

1.公司仓豆活动从2016年3月1日开始，本规则亦从当日开始生效。

2.随着公司仓豆活动的开展和情况的变化，公司将不定时地制定补充规则以对本规则进行更改和完善。

3.用户因参加公司开展的促销、奖励、赠与等活动而获得的仓豆，其具体条件和使用方式按照公司分别发布的活动公告、通知为准。

4.在法律允许范围内，本规则之最终解释权归公司所有。
