退换货政策
一．退换货规则

1. 为保障您的权益，请您收到货物后即与配送人员当面核对商品的种类、数量、规格、赠品等是否正确，确认无误后再进行签收。 在特殊情况下，您委托他人签收的行为视同于您本人已对商品进行签收。您对商品进行签收后，佰万仓将不再接受以上问题的投诉。 如有任何问题，请致电：400-021-9898（周一至周五9：30—18：30），客服人员会及时为您处理。
1）如外包装箱破损，请当面及时向配送人员提出，并及时通知我们，我们会处理并承担由此而产生的运输费用。如您签收有明显损坏迹象的外包装后再投诉货物有误或有损坏，恕我们后续不再受理该单的投诉。如货物外箱贴有易碎标识，请务必当场开箱检查箱内货物是否有破损。如有损坏,可直接在签收单上注明:“内件损坏,本人拒收”字样,由物流公司再返回佰万仓。如产品不符，您可拒收。
2）对于商品数量缺失的情况，我们只受理当场投诉，即货物配送到家后，客户当面清点货物数量并及时向配送员提出的投诉。一旦签收，我们将不受理此类投诉。
2. 佰万仓严格遵循国家相关法律法规，对佰万仓所售商品履行退货和换货的义务。
如符合以下条件，自顾客收到商品起7日内可申请退换货，超过时效，佰万仓不接受任何退换货申请：
1）商品出现国家相关法律法规中所规定的非人为质量问题；
2）在7日内至生产厂商的售后服务中心开具证明商品有质量问题的检测报告（如售后服务中心无法提供检测报告，则需提供商品检测或故障单）；
3）商品外包装完整未损坏，商品及其标配配件、赠品（如购买时有赠品）、说明书、保修凭证和发票均齐全（如申请退换货的商品为成套商品中的一个商品如礼盒、礼包，则需退回此整套商品）；
3. 以下情况佰万仓恕不提供退换货服务：
1）任何非由佰万仓出售的商品；
2）商品已使用（有质量问题除外）或非正常使用磨损；
3）商品外包装不完整或损坏，标配配件、赠品（如购买时有赠品）或说明书不完整，保修凭证和发票任一缺失或涂改，保修凭证上的的商品型号或序列号与商品实物不符；
4）擅自撕毁或涂改商品标贴、序列号、防伪标记；
5）未经授权的疏忽、误用、滥用、拆装、碰撞、进液、事故、改动、不正确的安装或保管而导致的商品质量问题；
6）由非生产厂商指定或授权维修点进行拆装或维修而导致的商品质量问题；
7）因行业商品的特殊性，对于博世、世达品牌全线商品、其他品牌非常规型号产品（原厂家定义，比如：非库存产品、定制、订货产品等）均不接受退换货，质量问题除外；
4. 佰万仓售后部门在收到顾客寄回的商品起15个工作日之内完成退换货申请的处理。特殊商品如有另行规定，以商品页面保修条款中的保修政策为准。
5. 在不违反国家相关法律法规的前提下，佰万仓对商品退换规则保留最终解释权。
6. 在第三方商家没有与客户达成另行约定的情况下，本退换货规则适用于佰万仓商城及所有的第三方商户。



二．退换货须知:

1. 如由佰万仓代开检测报告耗时过长可能会错过7日的退换货时间。为保证顾客售后权益，佰万仓建议顾客自行到厂商售后服务中心进行检测并开具检测报告。
2. 为保证顾客售后权益，佰万仓建议顾客如需要维修保养，请自行前往厂商售后服务中心处理。
3. 对于需要退换货的需求，佰万仓必须先收到该商品，并在检查核实无误后，再安排退换货。
4. 若已开具增值税发票，需寄回发票原件及退票说明，否则，不做退货、退款动作，且由此产生的损失由客户自行承担。
5. 由客户原因造成的退换货，运费由客户承担；由佰万仓原因造成的退换货，运费由佰万仓承担。
6. 不适用7日内无理由退换货的商品，包含但不限于以下：
1）客户定作的商品
2）其他根据商品性质在客户购买时确认不宜退货的商品。

8. 除了寄还地址以外，在第三方商家没有与客户达成另行约定的情况下，以上退换货须知同时适用于佰万仓商城及所有的第三方商户。



三．如何退款

佰万仓现接受微信、网银、银行转账三种支付方式，对于账期客户，另作安排。
原则上，从佰万仓收到退款申请及客户退回的产品之日起，将依据相关规定在一个月内退款给客户。但由于客户递交退款申请及商品退回的延迟，以及银行、微信等第三方支付平台办理退款周期的原因，不排除退款时间延迟的可能。
由退货产生的退款，佰万仓在收到所退货物并验收无误后，将直接退回到客户选用的微信、网银、银行转账的账号中。
（1）发票抬头为公司的均退款至公司账户，请提供和发票抬头一致的账户信息。
（2）发票抬头为个人的请提供和付款人信息一致的银行账户信息。
银行转账方式支付的订单，请务必保存好汇款单，退款时需提供汇款单；退款由佰万仓账户直接退款。
如无特别约定，所有第三方商户遵循与佰万仓商城相同的退款规则。
