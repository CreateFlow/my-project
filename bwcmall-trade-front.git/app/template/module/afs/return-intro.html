售后类型介绍
1、退货退款：
        适用于符合7天无理由退货政策或者买卖双方另行约定的情形，买家可以发起“退货退款”,详情请见退换货政策。
2、换货：
        适用于符合15天无理由换货政策或者买卖双方另行约定的情形，买家可以发起“换货”,详情请见退换货政策。
3、仅退款：
        适用于卖家还未发货但买家已经支付货款或者买卖双方另行约定的情形，买家可以发起“仅退款”。
