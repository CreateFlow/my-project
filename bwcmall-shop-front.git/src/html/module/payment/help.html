什么是账期支付
账期支付是指，买家在授信额度内下单后不用付款就可以进货，在最后还款日内完成支付的交易方式。
账期类型解释
周结
按每个自然周作为结算周期，到每周日的凌晨整点出上周的账单。
 
月结
按每个自然月作为结算周期，到每月最后一天凌晨整点出上月的账单。
双月结
按两个连续自然月作为结算周期，到双月最后一天凌晨整点出上两个月的账单。
使用流程
注意事项
1、卖家可以根据情况调整买家的账期类型、最后还款日。调整后将在下一期生效。
2、卖家可以根据实际情况提升买家的授信额度，提升额度立即生效。
3、卖家可随时停用买家享受的账期支付额度，取消后买家无法再使用账期支付。
4、最后还款日：卖家为每个买家设置最后的还款期限，超过该期限买家不支付账单，则买家不能再使用账期支付。
5、如果买家的信用额度已使用完毕，或者买家超过最后还款日没有支付账单，则买家不可以继续使用账期支付。
